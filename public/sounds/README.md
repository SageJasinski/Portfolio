# Sound Files Directory

Place your sound effect files here:

| Filename | Usage |
|----------|-------|
| projector-start.mp3 | Intro animation start |
| projector-loop.mp3 | Projector running sound |
| film-flicker.mp3 | Project hover effect |
| reel-load.mp3 | Opening project modal |
| whoosh.mp3 | Page transitions |
| click.mp3 | Button clicks |
| hover.mp3 | Navigation hover |

## Recommended Sources

- [Freesound.org](https://freesound.org) - Free sound effects
- [Pixabay Sounds](https://pixabay.com/sound-effects/) - Royalty-free
- Record your own for authenticity!

## Tips

- Keep files small (< 100KB each)
- Use MP3 format for best compatibility
- Subtle sounds work better than loud ones
- Test volume levels in the browser
