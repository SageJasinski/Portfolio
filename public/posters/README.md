# Project Posters Directory

Place your movie poster images here.

## Format Requirements

- **Aspect Ratio**: 2:3 (standard movie poster)
- **Recommended Size**: 600x900px minimum
- **Formats**: JPG, PNG, or WebP

## Naming Convention

Use lowercase with hyphens:
- `project-name.jpg`
- `awesome-film.png`

Then reference in `src/data/projects.js`:
```javascript
poster: '/posters/project-name.jpg'
```
